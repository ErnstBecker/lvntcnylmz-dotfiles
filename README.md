# Arch Linux Dotfiles

## Modules 
- Shell : fish
- Promt Theme Engine : oh-my-posh
- Window Manager : swaywm
- Status Bar : waybar
- System Info Tool : neofetch
- Terminal : terminator
- Display Manager : ly
- File Manager : ranger
- Text Editor: Nvim / VsCodium
- Window Switcher/App Launcher : rofi
- Spotify Command Line: spicetify
